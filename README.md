## Descripción

[Nest](https://github.com/nestjs/nest) API para gestión y venta de equipos informáticos

## Instalación

```bash
$ yarn install

$ yarn db:initialize
```

## Lanzar la aplicación

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
